## React-Music-Player
![首页](/pic.gif)
### 如何执行
####  1、将项目克隆到本地，cd到react-redux-todolist
```javascript
git clone git@github.com:chenjun1127/react-music-player.git
cd react-music-player
```
#### 2、安装依赖
```javascript
npm install
或
yarn install
```
#### 3、执行
```javascript
npm start
// npm run build(打包)
```
#### 4、打开浏览器浏览 http://localhost:3000/
